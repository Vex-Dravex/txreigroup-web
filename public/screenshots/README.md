# Screenshots Directory

This directory should contain screenshots of the website features for the pre-launch landing page carousel.

## Required Images

Please add the following screenshot images to this directory:

1. **deals.png** - Screenshot of the Off Market MLS / Deals page
2. **vendors.png** - Screenshot of the Vendor Marketplace / Contractors page
3. **submit-deal.png** - Screenshot of the Deal Submission form
4. **courses.png** - Screenshot of the Education Center / Courses page
5. **forum.png** - Screenshot of the Community Forum page
6. **transactions.png** - Screenshot of Transaction Services (or related feature)

## Image Specifications

- **Format**: PNG or JPG
- **Recommended Size**: 1920x1080px or larger (16:9 aspect ratio)
- **Quality**: High quality, clear screenshots
- **Optimization**: Consider compressing images for web performance

## Logo

The logo image should be placed in `/public/logo.png` or `/public/logo.svg`.

The logo file will be used in:
- The hero section of the landing page
- The header (if you choose to update it)
